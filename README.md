# Ltalk 即时通讯


这是一个非常简单的即时通讯软件, 客户端实现跨平台, 服务端采用c++ epoll 非阻塞模式实现处理大量并发, 传输协议为http, json数据格式来进行交互

你可以进入该网址: http://39.97.119.22 来浏览web接口
